# Pear VPN Telegram Bot (Customer Sales Flow)

Бот теперь работает как магазин VPN:
- клиент выбирает тариф;
- делает оплату;
- админ подтверждает оплату в Telegram;
- бот создаёт пользователя в панели и выдаёт готовый `vless://`.

## Быстрый запуск

1. Подготовь env:
   - `cp bot/.env.example bot/.env`
2. Укажи:
   - `TELEGRAM_BOT_TOKEN`
   - `PEAR_API_URL=http://127.0.0.1:5174`
   - `TELEGRAM_ADMIN_IDS=123456789` (кто подтверждает оплаты)
   - `PAYMENT_TEXT` (реквизиты/инструкция оплаты)
3. Установи зависимости:
   - `python3 -m venv bot/.venv`
   - `bot/.venv/bin/pip install -r bot/requirements.txt`
4. Запусти:
   - `bot/.venv/bin/python bot/main.py`

## Пользовательский сценарий

- `/start` -> кнопка "Купить VPN"
- выбор тарифа -> создаётся заказ
- клиент жмёт "Я оплатил(а)"
- админу приходит кнопка "Подтвердить оплату"
- после подтверждения бот:
  - создаёт пользователя через backend API,
  - синхронизирует Xray,
  - отправляет клиенту личный `vless://`.

## Команды

Для клиентов:
- `/start`

Для админов:
- `/admin_orders`
- `/health`

## Хранилище

Локальная БД заказов: `bot/bot.db`
